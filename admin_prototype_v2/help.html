Frequently Asked Questions

Quick answers to common questions

Navigate to the Knowledge Base section and click "Upload Documents". You can upload PDF, DOC, or TXT files up to 10MB. The system will automatically process and index your documents for the chatbot.

Use the Queries section to view all user interactions, response accuracy, and identify areas needing improvement. You can filter by date range, query type, and resolution status.

Currently supported formats include PDF, Microsoft Word (.doc, .docx), plain text (.txt), and markdown (.md) files. Each file should not exceed 10MB in size.

Navigate to Settings to adjust response tone, fallback messages, and conversation flow. You can also set up custom greetings and define response templates for common queries.
